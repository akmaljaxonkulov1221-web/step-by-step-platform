# Kwork Project: Ta'lim Platformasini To'liq Yakunlash

## Loyiha nomi: Ta'lim Platformasi - Student Management System

## Loyiha tavsifi:
Mavjud ta'lim platformasidagi bo'sh joylarni to'ldirish va student management funksiyalarini to'liq yakunlash kerak. Platform Flask + SQLAlchemy da yozilgan.

## Qilinishi kerak bo'lgan ishlar:

### 1. Student Management Interface ni To'ldirish
- Student ro'yxatidagi bo'sh joylarni to'ldirish
- Student qo'shish formasi to'liq ishlashi
- Student tahrirlash interfeysi
- Student o'chirish funksiyasi
- Parolni yangilash modallari

### 2. Admin Dashboard ni To'ldirish
- Statistika kartalarini to'liq ishlashi
- So'nggi test natijalari
- Haftalik test jadvali
- Guruhlar va fanlar ro'yxati

### 3. Certificate Management
- Sertifikat yuklash interfeysi
- Sertifikat ro'yxati
- Sertifikat o'chirish funksiyasi

### 4. Test Management
- Test yaratish va tahrirlash
- Savollarni boshqarish
- Test natijalarini ko'rish

### 5. UI/UX Yaxshilashlar
- Responsive design
- Bootstrap 5 komponentlarini to'g'ri ishlashi
- Modal oynalarni to'liq funksionalligi
- Form validatsiyalari

## Texnik talablar:

- SQLAlchemy database
- Bootstrap 5
- JavaScript (vanilla)
- HTML/CSS
