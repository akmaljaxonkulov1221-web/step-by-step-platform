## **Technical Information**

### **Technology Stack**
- **Backend:** Flask 2.3.3 with SQLAlchemy
- **Frontend:** Bootstrap 5.3 with FontAwesome
- **Database:** SQLite (cloud storage)
- **Deployment:** Render.com cloud platform
- **Language:** Uzbek (complete localization)

### **Key Features**
- **Authentication:** Secure login system
- **Database:** User, Group, Subject, Test models
- **API:** RESTful endpoints for all operations
- **Security:** Password hashing, session management
- **Backup:** Automated database backup system
